# 11-1向外行解释专业问题
核心：专业术语只是过渡，关键是说出专业术语的时机     
**不要一上来就说专业术语**，  
1.首先需要从日常生活切入，介绍该专业在日常生活中的应用或者现象，让外行有个基本印象（在介绍日常现象中最好能够体现你对外行的关心）；  
2.采用术语总结；  
3.结合术语和大白话展开解释    
从客户，读者，外行的立场，考虑怎么表达
# 11-2如何和小孩沟通
３个错误
１．用命令和胁迫的方式（危害了孩子的自主权）-->　找到双方共同的需要，让孩子觉得做一件事是处于自己的需要  
２．道德评判（直接告诉小朋友对错，相当于直接评判了行为）-->　基于请求的语言表达＂我们都住在一块，能不能不乱扔衣服，保持环境的整洁＂　引导孩子树立责任感  
３．物化人格（对待孩子像对待正常的人类）-->传达无条件的爱和尊重
# 11-3如何回应＂最近很忙＂
忙：　你的事没我现在的事重要（给你留面子）
错误的做法：  
１．那你什么时候有空呢？  
２．先问什么时候有空...  
正确做法：  
１．先说事情，再说时间，再问对方是否有时间？（给对方足够时间考虑，表达足够的尊重）  
２．事先给对方提供几个可选择的时间段  
想让对方来，增加吸引力；有个更吸引人的新版本；

# 11-5怎样向高人请教  
不要问标准答案式的问题  
激发对方兴趣,把别人当导师  
问得具体点,和自己生活相关  
"您在做重大决策时,有哪些小习惯呢"  
"在您像我这个年纪的时候,做科研这条路给您带来了什么,做科研苦不苦呢"  
"我和您一样当过老师,您觉得从事教育这个行业对创业有什么帮助呢"  
# 11-8聊天时如何找话题    
找大多数人没有分歧并且可以展开的话题
大多数人：影视，业余爱好
尽量避免分歧：不要问有比较的话题
容易展开：涉及背景，典故;发散性的思维，从一个问题到另一个话题的联想
聊名字破冰(名字常常包含美好的寄托）
学会扩展话题，讲落地的话题   
谈资（平时打好基础） 
# 11-9怎样说套话  
真实和直接都是选择性的  
1.适当的沉默(影响不大的价值冲突)  
2.适度的谎言    
3.暗示(您好,我刚刚看到角落那边写这边是无烟区,物业的保安一直盯着这边看,您看您要不要换个地方呢；街区那边有个新餐厅,你有去尝过么?)  
# 11-11把是非变成选择  
"老板,因为我最近我很多事,所以在品质和效能上可能没有之前那么好,您看这样ok么?"  
"价格便宜点可以,这要看能是更在意价格还是更主动品质了?"

让对方向内找理由,变得更加理性
